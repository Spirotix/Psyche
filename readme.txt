Pour compiler :

mettez vous dans le dossier PSYCHE_equipe_A2_B1
faites la commande
	javac @compile.list -d ./class

Pour excuter :
	cd ./class

pour le jeu de base :
	java Psyche

(Vous avez alors choix entre le contructeur de map et le jeu
Si aucun thème n'est choisi le thème de base est celui de l'age de psyche)

pour les scenarios :
	java Psyche scenario1.run

Si vous voulez le scénario 2 entrez
	java Psyche scenario2.run

(Pour parcourir les étapes d'un scénario il faut suivre les instructions données dans le terminal)
